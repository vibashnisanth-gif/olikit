## Observations
1. **US has most content depth** — Only locale with state-level content (10 states with 3 tools each = 30 state pages)
2. **UK has richest locale overrides** — All 7 tools get country-specific FAQs, AI answers, steps, and examples
3. **AU, CA, NZ, IN, SG** — Have overrides for 6 tools (excluding profit-margin-calculator and business-loan-calculator)
4. **Generic fallbacks exist** — Tools without locale overrides fall back to US-generated content with locale name substitution
5. **Non-US subregions defined but empty** — Australia (6 states), Canada (4 provinces), India (8 states) all have subregion definitions but ZERO state-specific content
6. **Examples use different base values** — Each locale uses realistic example salaries
