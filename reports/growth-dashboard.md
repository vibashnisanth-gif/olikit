## Weekly Review Process

1. **Monday**: Pull metrics from all sources
2. **Tuesday**: Analyze changes vs baseline
3. **Wednesday**: Identify top 3 optimization opportunities
4. **Thursday**: Implement optimizations
5. **Friday**: Deploy and verify

---

## Alert Thresholds

| Metric | Threshold | Action |
|--------|-----------|--------|
| Organic Clicks | -20% week-over-week | Investigate ranking drops |
| 404 Errors | >10 new | Fix broken links immediately |
| Build Failures | >1 per week | Fix CI immediately |
| CLS | >0.1 | Fix layout shifts |
| LCP | >3s | Investigate performance regression |
| Revenue | -15% week-over-week | Review ad placement and CTR |
